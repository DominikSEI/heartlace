3. Einladungs-Generator

Welche Daten werden verarbeitet?

Wenn Sie eine Einladung erstellen, geben Sie folgende Daten ein:

  • Namen des Brautpaars
  • Hochzeitsdatum, Uhrzeit, Ort
  • Persönliche Geschichte und Nachricht an Gäste
  • Stilpräferenzen und Farbauswahl
  • Optional: Fotos

KI-Verarbeitung via Anthropic API

Die von Ihnen eingegebenen Daten werden zur Generierung Ihrer Einladungswebsite an die Anthropic, PBC API (San Francisco, CA, USA) übermittelt. Anthropic verarbeitet diese Daten ausschließlich zur Erfüllung des Auftrags (Generierung der HTML-Seite) und speichert sie nicht dauerhaft. Weitere Informationen: anthropic.com/privacy

Rechtsgrundlage: Art. 6 Abs. 1 lit. b DSGVO (Vertragserfüllung).

Hosting der generierten Seiten

Fertige Einladungsseiten werden über die Netlify API auf deren Servern gehostet und sind unter einer eindeutigen URL für maximal 12 Monate nach dem Hochzeitsdatum öffentlich erreichbar. Danach werden sie automatisch gelöscht.

4. RSVP-Formular

Die über das RSVP-Formular auf Ihrer Einladungsseite eingehenden Daten der Gäste (Name, Personenanzahl, Nachricht) werden gespeichert und ausschließlich an das Brautpaar weitergeleitet. Die Gäste werden beim Ausfüllen des Formulars über diese Verarbeitung informiert.
